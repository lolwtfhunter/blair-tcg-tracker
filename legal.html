Map Data & Store Hunter

The Store Hunter feature uses map data and services from the following providers:

OpenStreetMap:
Map data © OpenStreetMap contributors. OpenStreetMap data is available under the Open Database License (ODbL). Map tiles are provided by the OpenStreetMap tile servers under the CC BY-SA 2.0 license.

Overpass API:
Store location data is queried from the Overpass API, a read-only API for OpenStreetMap data.

Nominatim:
Address and zip code geocoding is provided by Nominatim, the OpenStreetMap geocoding service.

Leaflet:
Interactive maps are rendered using Leaflet, an open-source JavaScript library licensed under the BSD 2-Clause License.

Store search results are best estimates based on available map data. Please verify store details, hours, and inventory before visiting.
